============================================
  GAME LOGOS - Instructions
============================================

Place game logo images in this folder.

NAMING CONVENTION:
- The filename must EXACTLY match the game name in the database.
- Use PNG or JPG format.

EXAMPLES:
  Minecraft.png
  Valorant.png
  League of Legends.png
  Genshin Impact.png
  Roblox.png
  Tekken.png
  Counter-Strike 2.png
  Dota 2.png
  Fortnite.png
  Apex Legends.png
  PUBG.png
  Overwatch 2.png
  Rocket League.png
  Street Fighter 6.png

RECOMMENDED SIZE:
- 64x64 pixels or 128x128 pixels (square)
- PNG with transparent background preferred

If no logo file is found for a game, a generic placeholder icon will be shown.
